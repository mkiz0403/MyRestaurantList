# 나의 맛집 리스트

## 서비스 설명
 유저별 맛집 리스트를 작성하고, 방문 횟수를 카운팅 하여 개인의 음식 성향을 알 수 있다.


무엇을 먹어야 할지 고민이 있을 때, 맛집 리스트 중 몇 개를 선정하고 룰렛을 돌려 랜덤으로 맛집을 선정할 수 있다.

다른 유저와 맛집을 공유할 수 있다.

## 기능 설명
### 1. 유저별 맛집 리스트 작성
+ 1-1 지도에서 맛집을 선택하여 카테고리별로 자신의 리스트에 등록할 수 있다.

+ 1-2 등록된 맛 집중 방문한 곳의 횟수를 카운팅 할 수 있고, 시간을 기록할 수 있다.

### 2. 맛집 랜덤 뽑기(선택 장애 해소)
+ 2-1 리스트에 등록되어 있는 맛집을 몇 개 선택해서 랜덤으로 맛집을 선택할 수 있다.


### 3. 다른 유저와 맛집 리스트 공유
+ 3-1 맛집 공유를 하면 지도에 공유된 맛집이 표시되고, 다른 유저들도 볼 수 있다.
